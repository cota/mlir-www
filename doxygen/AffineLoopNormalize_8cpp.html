static void normalizeAffineFor ( AffineForOp  op)
static

Normalizes affine.for ops.

If the affine.for op has only a single iteration only then it is simply promoted, else it is normalized in the traditional way, by converting the lower bound to zero and loop step to one. The upper bound is set to the trip count of the loop. For now, original loops must have lower bound with a single result only. There is no such restriction on upper bounds.
